# ✨ Características Principales (Features)

El **Gestor de Flujo de Trabajo** ha sido diseñado para proporcionar las herramientas necesarias para una gestión ágil, facilitando la organización del equipo. 

A continuación se destacan las funcionalidades principales:

- **Gestión Visual de Tareas (Tablero estilo Kanban):** Interfaz sencilla que permite mover distintas tareas ("To Do", "In Progress", "Done").
- **Asignación y Roles de Usuarios:** Posibilidad de que existan distintos perfiles que puedan editar, añadir información o simplemente supervisar.
- **Despliegue Sencillo:** Estructurado para que usando Docker cualquiera pueda levantar todo el sistema (Front, Back y BD) en segundos.
- **Independencia Frontend/Backend:** La lógica profunda del sistema está separada de su presentación visual.
