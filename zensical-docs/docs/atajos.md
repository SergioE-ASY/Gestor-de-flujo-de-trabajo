# Atajos de teclado

E-asy incluye atajos de teclado para navegar y crear contenido sin usar el ratón. Los atajos no se activan cuando el foco está en un campo de texto, selector o editor.

## Panel de ayuda

Pulsa `?` en cualquier página para abrir el panel con todos los atajos disponibles.

## Paleta de comandos

| Atajo | Acción |
|-------|--------|
| `Ctrl + K` / `⌘ K` | Abre la paleta de comandos |
| `Escape` | Cierra la paleta o cualquier modal abierto |

La paleta de comandos permite buscar y navegar a cualquier sección de la aplicación sin salir del teclado. Empieza a escribir para filtrar resultados.

## Navegación global (prefijo `g`)

Los atajos de dos teclas se activan pulsando primero `g` y luego la segunda tecla en menos de 1 segundo:

| Secuencia | Destino |
|-----------|---------|
| `g` → `d` | Dashboard |
| `g` → `t` | Mis tareas |
| `g` → `o` | Mis organizaciones |
| `g` → `p` | Mis proyectos |
| `g` → `r` | Roadmap del proyecto actual |

## Navegación por pestañas (prefijo `g`)

Desde la vista de detalle de un proyecto:

| Secuencia | Pestaña |
|-----------|---------|
| `g` → `b` | Tablero (Board) |
| `g` → `l` | Lista |

## Acciones rápidas

| Atajo | Acción |
|-------|--------|
| `c` | Crear nueva tarea (si la página tiene el enlace disponible) |
| `?` | Abrir panel de atajos |

!!! note "Campos de texto"
    Los atajos de una sola tecla (`c`, `?`, `g`…) se ignoran automáticamente cuando el cursor está dentro de un campo de texto, área de texto, selector o el editor de Markdown.
