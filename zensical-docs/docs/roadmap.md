# Roadmap

El Roadmap es una vista tipo Gantt que muestra todas las tareas del proyecto con fechas en una línea de tiempo. Permite ver de un vistazo el plan de trabajo, los solapamientos y las dependencias entre tareas.

**Ubicación:** Botón **Roadmap** en la barra superior del proyecto (junto a las pestañas).

## Controles de zoom

| Opción | Descripción |
|--------|-------------|
| **Semana** | Cada columna representa un día. Vista detallada a corto plazo. |
| **Mes** | Cada columna representa una semana. |
| **Trimestre** | Cada columna representa un mes. Vista general del proyecto. |

Pulsa **Ir a hoy** para centrar la vista en la fecha actual.

## Línea "Hoy"

Una línea vertical roja marca el día actual en el gráfico, visible en el modo Semana y Mes.

## Barras de tarea

Cada tarea con fechas de inicio y fin aparece como una barra horizontal coloreada según su prioridad:

| Color | Prioridad |
|-------|-----------|
| Verde | Baja |
| Azul (accent) | Media |
| Naranja | Alta |
| Rojo | Crítica |

Pulsar una barra abre el detalle de la tarea.

Las tareas sin fechas de inicio o fin **no aparecen** en el Roadmap. El aviso "Ninguna tarea tiene fechas" se muestra si ninguna tarea del proyecto tiene fechas asignadas.

## Agrupación por sprint

Las tareas se agrupan bajo su sprint correspondiente. Las filas de sprint aparecen como encabezados de sección con fondo diferenciado.

## Dependencias

Activa o desactiva el checkbox **Dependencias** en la barra de herramientas para mostrar las líneas de dependencia entre tareas relacionadas.

!!! tip "Añadir fechas"
    Si el Roadmap aparece vacío, edita las tareas y añade fechas de **Inicio** y **Vencimiento** para que aparezcan en la línea de tiempo.
